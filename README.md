# AI Daily Lab

An automated AI-powered daily task system for Data Science, Machine Learning and AI.

## What this does
- Generates daily DS/ML tasks using Gemini
- Avoids duplicates
- Tracks progress
- Creates monthly reports
- Runs automatically with GitHub Actions

## Stack
- Python
- Gemini API
- GitHub Actions
- JSON logging
- CI automation

## Outputs
- output/task.md
- output/today.json
- output/stats.json
- output/streak.json
- reports/monthly/YYYY-MM.json

## Use case
Practice Data Science and AI every day with guided automation.
